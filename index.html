Explain DOM Traversal

DOM (Document Object Model) is a tree like structure hierarchy that represents HTML elements. DOM Traversal is a concept in which we navigate and manuplate HTML elements using DOM.




This is the basic DOM version of a HTML code, In DOM traversal we navigate and access any of the element using multiple ways and manuplate elements in runtime.

Difference between document.children and Other Selection Methods:

There are many ways of selecting or navigating thoughout the elements, document.children returns a HTMLcollection in which there are only HTML elements while in the otherways the the result could be different.
For Example:

  • document.body.children : Returns All HTML elements.
  • document.body.childNodes : Returns all Nodes icluding text and comment.
  • document.body.firstChild : Returns only first childNode wheather it is a text node or an Element.
  • document.body.firstElementChild : Returns only first HTML Element

Difference Between append and appendChild Method

Difference between append & appendChild is if we use appendChild it becomes a childNode of that particular element we want to add with and we cant add otherthan an element, while using append we can also append a string.

Differences Between append, insertBefore, and insertAdjacentHTML

using append we can add strings to an element,
insertBefore is used when we want to add an element on a a prticular place,and
insertAdjacentHTML have four parameters

Difference Between insertAdjacentHTML, insertAdjacentText, and insertAdjacentElement

  • insertAdjacentHTML :Used to add HTML code | i.e: Tags.
  • insertAdjacentText :Used to create a textnode and add to particular place.
  • insertAdjacentElement : Also used to add an element to a given postition.

Differences Between document.getElementById, tagName, className, querySelector, and querySelectorAll:

  • document.getElementById:Select an element by its idName in DOM.

  • tagName:Directly select Elements with their name also called typeSelector in CSS.

  • className:Selects an element with className.

  • querySelector: Uses CSS Properties to select elements in DOM.

  • querySelectorAll:Also use CSS properties to select all elements with a particuler name or class
